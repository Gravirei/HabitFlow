# HabitFlow - Tech Stack

## Language & Frameworks
- **Language:** TypeScript - For strong type safety and improved developer experience.
- **Frontend Framework:** React 18 - Powering the interactive UI with modern hooks and concurrent features.
- **Build Tool:** Vite - Lightning-fast development server and optimized build processes.

## Styling & UI
- **Styling:** Tailwind CSS - Utility-first CSS for rapid and consistent styling.
- **Components:** Custom React components following the design guidelines.
- **Animations:** Framer Motion - For high-quality, fluid UI transitions.

## State Management & Routing
- **Global State:** Zustand - Lightweight and performant state management with persistence.
- **Routing:** React Router v6 - Declarative routing for a smooth single-page application experience.
- **Forms:** React Hook Form & Zod - For efficient form management and robust schema validation.

## Backend & Storage
- **Backend-as-a-Service:** Supabase - Providing secure authentication, real-time database, and cloud storage.
- **Mobile Support:** Capacitor - Enabling cross-platform mobile deployment (Android/CLI).

## Testing & Quality
- **Unit & Integration Testing:** Vitest and React Testing Library - For fast and reliable testing.
- **E2E Testing:** Playwright - Ensuring core user flows are correct across browsers.
- **Linting & Formatting:** ESLint and Prettier - Maintaining code quality and style consistency.

## Utilities
- **Date Handling:** date-fns - For modern and lightweight date manipulation.
- **Utilities:** clsx and tailwind-merge - For flexible and collision-free class merging.
